# EPIP (Elements of Programming Interviews in Python) Code Snippets Cheat Sheet

**EPIP (Elements of Programming Interviews in Python)** Code Snippets Cheat Sheet to refresh/skim before coding interview at FAANG (Facebook, Apple, Amazon, Netflix, Google):

1. **Basics**

    *1.1 Shortcuts* (values swap, tuple/list/dictionary unpacking, strings slicing)

    *1.2 Arithmetic* (integer division, modulus, exponentiation)

    *1.3 Min/max*

    *1.4 Lambda funcitons* (anonymous functions)

    *1.5 Ternary operator* (like in JavaScript "condition ? value_if_true : value_if_false")
2. **Loops** (lists, dictionaries, sets, zip)
3. **List processing** (push/pop, accumulate/prefix sum, count, functional programming (map (built-in), filter (built-in), reduce (functools)), comprehensions, sort)
4. **Conditions** (all, any)
5. **Tuples** (list of tuples, unpacking, sorting tuples, min/max, sort with lambda funciton)
6. **Heap** (heapifq, heapify, heappush, heappop, nlargest, nsmallest)
